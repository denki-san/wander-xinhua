# Shangsheng Xinsuo modeling reference

## 地理定位

- 上生·新所位于延安西路 1262 号，OSM 场地边界为 way `765939973`。
- 本地原始快照：`docs/research/data/shangsheng-xinsuo-overpass-20260716.json`。
- 场地内独立建筑轮廓按同一新华路街道投影生成，不在 JSX 中手填世界坐标。

## 必须可识别的建筑

1. 海军俱乐部及露天泳池：白色两层连续拱廊、浅米色内墙、深色门窗、蓝绿色长方形泳池、池沿马赛克与暖色廊灯。
2. 哥伦比亚乡村俱乐部：两层米白/浅灰外墙、红褐瓦坡屋顶、首层连续圆拱、二层规则窗列与中央装饰山花。
3. 孙科别墅：暖灰褐粗粝墙面、红褐西班牙瓦、多层错落坡屋顶、尖拱入口、圆角塔楼、前庭草坪与低矮花篱。
4. 工业改造建筑：保留浅米、红砖、灰褐墙面和连续大窗，局部加入锯齿/低坡屋顶、外露深色框架与当代玻璃入口。
5. 二期建筑：以灰白石材、深色金属、玻璃和内院式体量表达新旧对照，不复制商业品牌。

## 关键环境要素

- 黑色金属入口雨棚与金色字形气质；运行时只显示“上生·新所”，不复制官方 logo。
- 大树、绿篱、香樟广场、石材铺地、庭院灯、长椅与咖啡桌。
- 泳池与孙科别墅花园是两处视觉核心；两处公开 OSM fountain 也保留。
