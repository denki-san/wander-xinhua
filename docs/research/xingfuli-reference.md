# 幸福里区块建模基准

## 目标与边界

- 建模对象为上海市长宁区番禺路 381 号、幸福路 67 号之间的 M+ 幸福里。
- 微信材料只说明新华路街区项目背景，不参与幸福里的视觉、颜色或空间判断。
- 视觉语言继续使用当前项目与 Messenger 的手绘低多边形风格，不做 Minecraft 方块风。
- 不显示门牌、店名、品牌标志，不把地图或公开照片作为运行时贴图；所有模型均由代码生成。

## 公开证据

1. [上海商报 / 新浪转载](https://k.sina.com.cn/article_7517400647_1c0126e47059031uwk.html)记载：场地前身为上海橡胶制品研究所，改造后形成连接幸福路与番禺路、约 200 米长的直线型开放街巷；文章同时描述了 7 栋独立建筑、垂直绿化停车墙、水池中的树和水塔屋顶花园。
2. [OpenStreetMap way 400066625](https://www.openstreetmap.org/way/400066625)给出东西向狭长场地轮廓与番禺路、幸福路之间的拓扑关系。本项目只使用其道路方向和建筑带关系，不把地图图块打包进产品。
3. [SmartShanghai 场地页](https://www.smartshanghai.com/venue/17729/xingfu_li_fanyu_lu)确认场地位于番禺路 381 号、近幸福路，并保留了 B、E、F 等座号的公开记录。
4. 2026-07-16 使用 Kimi WebBridge 1.11.3 在真实浏览器中核对百度地图公开页面。搜索结果可见 A-F 与 7 座，但 POI 名称不足以把座号可靠对应到每个建筑轮廓，因此代码只使用中性方位 ID。公开外观图补充确认了番禺路入口的垂直绿墙、白色玻璃转角楼、一排四棱矮石桩、内部灰色步行铺地，以及另一入口的彩色墙绘。照片只用于人工观察，不进入代码或部署产物。
5. 多角度公开照片共同显示：主巷两侧以白色抹灰、银灰金属板、蓝灰玻璃和深灰框架为主；局部存在珊瑚色体块、木格栅、黑色玻璃屋顶亭；中段有长条倒影池、喷泉小品、池中树、木桥、红色遮阳伞、盆栽与户外座椅。

## 本轮细节提升门槛

- 改造前基线：`131` 个 JSX mesh 标签、`65` 个 geometry 标签，记录于 `model-detail-baseline.json`。
- 不以无意义细分凑面数；新增内容必须能在上表照片中找到对应。
- 本轮至少把下列可辨识构件补成原有数量的两倍：入口框架与矮桩、立面窗框/竖梃、店面雨棚、空调与落水管、池沿压顶与溢水槽、桥板与扶手、铺地分缝、灯具/座椅/盆栽。
- 位置仍由 OSM way `400066625` 的场地中心、长轴和轮廓控制，细节不得改变建筑占地或把碰撞体扩大到檐口、绿墙与街具之外。

## 空间与比例清单

| 对象 | 场景实现值 | 判断依据 | 精度等级 |
| --- | ---: | --- | --- |
| 东西向主巷 | 长 94、建筑净距约 12 | OSM 拓扑、入口和巷内长焦照片 | 高（相对比例） |
| 建筑组 | 北侧 4 个主要体量、南侧 3 个主要体量 | 公开报道的 7 栋、OSM 建筑带轮廓与公开照片 | 中（体量归并） |
| 建筑高度 | 2-4 层，每层 2.08 | 立面窗格、人物与栏杆尺度对照 | 中 |
| 倒影池 | 长 18、宽 2.9 | 多张巷内照片，以铺地模块和座椅作参照 | 中高 |
| 番禺路入口 | 绿墙与北侧白色玻璃转角楼夹出入口，四枚矮石桩 | 百度地图公开外观图 | 高（视觉结构） |
| 西侧入口 | 冷灰建筑端墙 + 抽象彩色墙绘 | 百度地图公开外观图 | 中高 |

这里的“精度”是面向 WebGL 场景的相对比例，不是测绘级尺寸。公开资料没有施工图、立面测绘或完整的院内街景，因此绝对高度、局部开间与设施间距只能由多张照片交叉估算；代码中不得把这些估算表述为真实测量值。

## 核心街具

- 长条倒影池：深色石材池沿、蓝绿色浅水面、四组低喷泉/石景、池中树与横跨水池的木桥。
- 番禺路入口：整面垂直绿化、深色基座、四棱锥矮石桩。
- 灯具：深灰细杆短臂街灯，沿巷两侧交错布置。
- 停留设施：木面深灰脚长椅、红色遮阳伞、桌椅与不同尺寸的深色花盆。
- 屋顶：深色玻璃亭、围栏平台、水塔及绿化盒子。

## 验收要求

- 从番禺路方向看，第一眼必须识别出“绿墙 + 白色玻璃转角楼 + 矮石桩”的入口组合。
- 进入街巷后，七栋体量、层数、冷暖颜色和两侧排列关系必须保持稳定，不能退回通用里弄建筑随机排布。
- 中段必须同时看见倒影池、池中树、喷泉石景与木桥；街灯、长椅、红伞和盆栽不可缺失。
- 场景只有原有的“一平米行动”一个可交互行动点；幸福里是闲逛空间，不新增任务或故事。
- 桌面 1440 × 900、手机 390 × 844 均需检查入口、主巷和近景；生产构建不得请求任何地图或照片资源。

## 2026-07-16 本地验收记录

- 北侧窗格已按南向立面翻转玻璃层，修复玻璃进入墙体后显示为黑块的问题。
- 番禺路入口已补白色玻璃转角体量、东端包角玻璃、上层退台和栏杆；绿墙贴合南侧建筑并采用双面植被。
- 角色碰撞边界与四面可见围墙内表面对齐；倒影池、绿墙、长椅和两组咖啡桌设置简化碰撞。
- 31 条铺地带合并为 2 个实例批次，144 个绿墙植物合并为 6 个实例批次；六盏路灯仅保留两盏局部 PointLight。
- 本机无头 Chrome 在 1440 × 900 与 390 × 844 视口各连续采样 5 秒，分别为 60.2 FPS 与 60.1 FPS。该结果用于发现明显回退，不等同于所有真实手机的性能保证。
- 浏览器 error 列表为空；控制台只有 Three.js `Clock` 与 `PCFSoftShadowMap` 的依赖弃用警告，没有本次改动引入的运行时异常。
