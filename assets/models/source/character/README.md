The runtime keeps the original rig, embedded 1024px atlas, 6,377 triangles,
54 nodes, 12 meshes and all 76 animation clips. Weapon and fantasy-cape nodes
are hidden at runtime; no source geometry is destructively removed. The existing
local messenger backpack is layered onto the rigged body to preserve the
project's visual identity.

The GLB is importable in Blender and is the editable source of truth used by the
application. `kaykit-rogue-preview.png` is the upstream author preview retained
for visual comparison.
